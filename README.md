# AFRINEX CMS

Application web développée en PHP permettant la gestion centralisée du contenu du site AFRINEX à travers une interface d'administration sécurisée.

## Fonctionnalités

* Authentification des administrateurs
* Gestion des pages et contenus
* Gestion des services
* Gestion des réalisations
* Gestion des témoignages
* Gestion des partenaires
* Gestion des formulaires de contact
* Téléversement de fichiers et d'images

## Technologies

* PHP
* MySQL
* HTML5
* CSS3
* JavaScript
* Bootstrap 5

## Structure du projet

```text
admin/          Interface d'administration
controllers/    Logique métier
includes/       Configuration et fonctions communes
public/         Point d'entrée et ressources publiques
templates/      Modèles de pages
uploads/        Fichiers téléversés
```

## Installation

1. Cloner le dépôt.
2. Importer le fichier `afrinex_cms.sql`.
3. Configurer les paramètres de connexion dans le fichier `.env`.
4. Déployer le projet sur un serveur PHP compatible.
